Custom WordPress Theme for ADVANCE-IT Group
This repository contains the source code for a bespoke WordPress theme developed for ADVANCE-IT Group, an IT solutions and development company. The theme is built from the ground up to provide a unique, branded experience and a fully manageable content structure.

Project Overview
The primary goal of this project was to create a modern, responsive, and professional website that reflects the identity of ADVANCE-IT Group. The theme is designed to be easily managed by administrators through the WordPress dashboard, separating the presentation layer from the content.

Key Features
Custom Design: A unique design tailored specifically to the branding and needs of the company.
Responsive Layout: Built using the Bootstrap framework, the theme is fully responsive and optimized for a seamless experience across desktops, tablets, and mobile devices.
Dynamic Content with Advanced Custom Fields (ACF): The theme heavily relies on the Advanced Custom Fields plugin. This allows administrators to easily update text, images, and other content on pages like the Homepage, About Us, and News without needing to write any code.
Custom Page Templates: Includes several custom page templates for different sections of the site, such as:
page-About.php
page-contact.php
page-news.php
Interactive Elements: Features custom JavaScript for interactive elements like modals and animations to improve user engagement.
Custom CSS Animations: Utilizes a dedicated animations.css stylesheet for fade-in and other visual effects to create a more dynamic feel.
Technologies Used
Backend: PHP, WordPress
Frontend: HTML5, CSS3, JavaScript
Frameworks & Libraries:
Bootstrap: For the responsive grid system and base components.
WordPress Plugins:
Advanced Custom Fields (ACF): For creating the flexible and manageable content fields.
Installation and Setup
To use this theme on a WordPress site, follow these steps:

Download the Theme: Clone or download the medae quere directory from this repository.
Upload to WordPress: Place the medae quere folder into the wp-content/themes/ directory of your WordPress installation.
Install Required Plugins:
From the WordPress admin dashboard, navigate to "Plugins" > "Add New".
Search for, install, and activate the "Advanced Custom Fields" plugin.
Activate the Theme:
In the WordPress admin dashboard, go to "Appearance" > "Themes".
Find the "Madae Quere" theme and click "Activate".
Populate Content: Use the custom fields provided by ACF on each page to add and manage the website's content.
